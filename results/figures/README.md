# Figures Directory

This directory contains visualizations generated by the Jupyter notebook.

## Generated Figures

After running the notebook, the following figures will be created:

- `convergence_final.png` - Convergence curves for all 8 functions over 13 rounds
- `breakthroughs.png` - Highlighted breakthrough functions (F1, F4, F5)

## Generating Figures

Run all cells in `notebooks/bbo_optimization.ipynb` to generate these visualizations.
